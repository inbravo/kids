0
1
2

Q5 Easy

x = 1
x = −1

x = 4
Hint: x² − 4 = 0 ⟹ x = ±2, so x = 2 is a zero.

Q7 Easy

Q16 Hard

Hint: (x+y+z)² = x²+y²+z² + 2(xy+yz+zx) ⟹ 81 = (x²+y²+z²) + 52 ⟹ x²+y²+z² = 29.
Q17 Hard

Q19 Hard
If p = 2 − a, show that a3 + 6ap + p3 − 8 = 0. Which identity explains this?

(a−b)3 identity; value = 0
Remainder Theorem; value = 8
Hint: Since a + p + (−2) = 0, the identity a³+p³+(−2)³−3ap(−2)=0 applies directly.
