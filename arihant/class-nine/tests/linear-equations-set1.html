Q1Easy
A linear equation in two variables is of the form:
ax² + by = c
ax + by + c = 0
ax + b = 0
axy + b = 0
Hint: The standard form is ax + by + c = 0 where a and b are not both zero, and the highest power of each variable is 1.
Q2Easy
Which of the following is a solution of 2x + y = 7?
(1, 4)
(2, 3)
(3, 2)
(0, 5)
Hint: Substitute each pair: 2(2) + 3 = 4 + 3 = 7 ✓. Check: 2(1)+4=6, 2(3)+2=8, 2(0)+5=5.
Q3Easy
The equation x = −4 represents:
A line parallel to the x-axis
A line parallel to the y-axis
A line through the origin
A diagonal line
Hint: x = constant (any number) is always a vertical line, which is parallel to the y-axis, cutting the x-axis at x = −4.
Q4Easy
How many solutions does the equation 3x + y = 10 have?
Exactly one
Exactly two
No solution
Infinitely many
Hint: For every value of x you choose, you get a corresponding y = 10 − 3x. Since x can be any real number, there are infinitely many solutions.
Q5Easy
The equation y = 5 represents a line:
Parallel to the y-axis
Parallel to the x-axis
Through the origin
A diagonal line
Hint: y = constant is a horizontal line — parallel to the x-axis, cutting the y-axis at y = 5.
Q6Easy
The graph of every linear equation in two variables is a:
Circle
Parabola
Straight line
Hyperbola
Hint: Degree 1 in both variables always gives a straight line. Parabola needs degree 2; circle and hyperbola are even more complex curves.
Q7Easy
Which of the following equations has (0, 0) as a solution?
2x + 3y = 6
x + y = 1
3x − 2y = 0
x − y = 5
Hint: Substitute (0,0): 3(0) − 2(0) = 0 ✓. All equations with c = 0 (ax + by = 0) pass through the origin.
Q8Easy
If (2, k) is a solution of x + 3y = 11, the value of k is:
1
2
3
4
Hint: Substitute x = 2: 2 + 3k = 11 → 3k = 9 → k = 3.

Q13Medium

8
9
10
12
Hint: From (a, 2): a + 2 = 5 → a = 3. From (−2, b): −2 + b = 5 → b = 7. So a + b = 3 + 7 = 10.

Q15Hard
The cost of a notebook is twice the cost of a pen. Write this as a linear equation (x = notebook cost, y = pen cost). If a pen costs ₹5, what does a notebook cost?
x = 2y; notebook costs ₹10
y = 2x; notebook costs ₹10
x + y = 2; notebook costs ₹10
2x = y; notebook costs ₹5
Hint: "Notebook is twice the pen" → x = 2y. At y = 5: x = 2 × 5 = 10. Standard form: x − 2y = 0.
Q16Hard
For what value of k does the equation 2x + ky = 6 have (1, 2) as a solution?
k = 1
k = 2
k = 3
k = 4
Hint: Substitute x = 1, y = 2: 2(1) + k(2) = 6 → 2 + 2k = 6 → 2k = 4 → k = 2.
Q17Hard
Where does the line 3x − 2y = 6 cross (i) the x-axis, and (ii) the y-axis?
(2, 0) and (0, −3)
(0, 2) and (−3, 0)
(3, 0) and (0, −2)
(2, 0) and (0, 3)
Hint: x-intercept (y=0): 3x = 6 → x = 2 → (2, 0). y-intercept (x=0): −2y = 6 → y = −3 → (0, −3).
Q18Hard
If (3, a) lies on 5x − 4y = 7, find a. Also, does (−1, −3) lie on the same line?
a = 2; yes, (−1, −3) lies on the line
a = 3; yes, (−1, −3) lies on the line
a = 2; no, (−1, −3) does not lie on the line
a = 1; yes, (−1, −3) lies on the line
Hint: 5(3) − 4a = 7 → 15 − 4a = 7 → a = 2. Check (−1, −3): 5(−1) − 4(−3) = −5 + 12 = 7 ✓.
Q19Hard
A taxi charges ₹20 as a fixed base fare plus ₹12 per km. Express this as a linear equation (x = km, y = total fare). What is the total fare for 8 km?
y = 12x + 20; ₹116
y = 20x + 12; ₹172
y = 12x − 20; ₹76
y = 10x + 20; ₹100
Hint: Total fare = (rate per km × distance) + base fare = 12x + 20. At x = 8: y = 96 + 20 = 116.
Q20Hard
On a coordinate plane, the lines x = 3, x = −3, y = 2 and y = −2 form a rectangle. What are the four vertices and what is the area?
(3,2), (3,−2), (−3,2), (−3,−2); Area = 24 sq. units
(3,3), (−3,3), (−3,−3), (3,−3); Area = 36 sq. units
(2,2), (2,−2), (−2,2), (−2,−2); Area = 16 sq. units
(3,2), (3,−2), (−3,2), (−3,−2); Area = 12 sq. units
Hint: Width = 3 − (−3) = 6, Height = 2 − (−2) = 4. Area = 6 × 4 = 24 sq. units. Vertices are all four intersections of the two vertical and two horizontal lines.
