Q1Easy
The equation y = mx (where m ≠ 0) always passes through:
(1, 0)
(0, 1)
(0, 0)
(1, 1)
Hint: Substitute x = 0: y = m × 0 = 0. So (0, 0) always satisfies y = mx regardless of the value of m.
Q2Easy
Which of the following represents the y-axis?
y = 0
x = 0
x = y
x + y = 0
Hint: The y-axis is the set of all points where x = 0. The x-axis is y = 0. Don't confuse the two!
Q3Easy
The equation 5 = 2x (treated as a linear equation in two variables x and y) has:
A unique solution
Infinitely many solutions
No solution
Exactly two solutions
Hint: Rewrite as 2x + 0·y = 5. Since y doesn't appear, y can be any value while x = 5/2 — giving infinitely many solutions (5/2, 0), (5/2, 1), (5/2, 2), …
Q4Easy
The point (−2, 5) lies in which quadrant of the coordinate plane?
Quadrant I
Quadrant II
Quadrant III
Quadrant IV
Hint: x is negative, y is positive → Quadrant II (top-left). Remember: I (+,+), II (−,+), III (−,−), IV (+,−).
Q5Easy
If x + y = 0, then y equals:
x
−x
0
1
Hint: x + y = 0 → y = −x. The graph is a straight line through the origin with slope −1, passing through (1,−1), (2,−2), etc.
Q6Easy
The equation of the x-axis is:
x = 0
y = 0
x = y
x + y = 0
Hint: Every point on the x-axis has y = 0, e.g. (1,0), (−3,0), (0,0). So the equation of the x-axis is y = 0.
Q7Easy
Which of the following is NOT a linear equation in two variables?
2x + 3y = 7
x − y = 0
x² + y = 5
3x = 5 − 2y
Hint: A linear equation can have degree at most 1 in each variable. Option C has x², making it degree 2 — not linear.
Q8Easy
If (k, −1) satisfies 3x − y = 7, the value of k is:
1
2
3
4
Hint: Substitute y = −1: 3k − (−1) = 7 → 3k + 1 = 7 → 3k = 6 → k = 2.

Q9Medium
The cost of 3 apples and 2 bananas is ₹24. Write as a linear equation (x = apple price, y = banana price). If one apple costs ₹6, find the cost of one banana.
3x + 2y = 24; banana costs ₹3
2x + 3y = 24; banana costs ₹4
3x + 2y = 24; banana costs ₹4
x + y = 24; banana costs ₹18
Hint: 3x + 2y = 24. At x = 6: 18 + 2y = 24 → 2y = 6 → y = 3. One banana costs ₹3.
Q10Medium
The graph of 2x + y = 6 cuts the y-axis at which point?
(3, 0)
(0, 3)
(0, 6)
(6, 0)
Hint: The graph cuts the y-axis where x = 0. Substitute: 2(0) + y = 6 → y = 6. So the point is (0, 6).
Q11Medium
Find the value of m so that (m, 3m) satisfies 4x − y = 5.
m = 1
m = 2
m = 5
m = 4
Hint: Substitute x = m, y = 3m: 4m − 3m = 5 → m = 5.
Q12Medium
How many integer-coordinate points lie on the graph of y = 3x − 1 for 0 ≤ x ≤ 2?
1
2
3
4
Hint: x=0: y=−1 → (0,−1) ✓; x=1: y=2 → (1,2) ✓; x=2: y=5 → (2,5) ✓. All three give integer y-values.
Q13Medium
The equation kx + 3y = 6 passes through (2, 0). Find k.
k = 1
k = 2
k = 3
k = 6
Hint: Substitute x = 2, y = 0: k(2) + 3(0) = 6 → 2k = 6 → k = 3.
Q14Medium

Q15Hard
Two students scored a total of 95 marks. The first scored 5 more than the second. Find both scores.
50 and 45
48 and 47
55 and 40
52 and 43
Hint: Let scores be x and y: x + y = 95 and x = y + 5. Substitute: (y+5) + y = 95 → 2y = 90 → y = 45, x = 50.
Q16Hard
The line 2x + py = 10 passes through (1, 4) and (q, 2). Find p and q.
p = 2, q = 3
p = 3, q = 2
p = 2, q = 4
p = 4, q = 3
Hint: At (1,4): 2(1) + 4p = 10 → p = 2. At (q,2): 2q + 2(2) = 10 → 2q = 6 → q = 3.
Q17Hard
A car travels from town A at 60 km/h. Write a linear equation for distance d in terms of time t (hours). How far has it travelled in 2.5 hours?
d = 60t; 150 km
t = 60d; 150 km
d = 60t; 120 km
d = 60 + t; 62.5 km
Hint: Distance = Speed × Time → d = 60t. At t = 2.5: d = 60 × 2.5 = 150 km.
Q18Hard
The sum of the digits of a two-digit number is 9. If the tens digit is x and units digit is y, write a linear equation. If x = 4, find y and the number.
x + y = 9; y = 5; number = 45
x − y = 9; y = 5; number = 40
xy = 9; y = 9/4; number ≈ 42
x + y = 9; y = 5; number = 54
Hint: Sum of digits: x + y = 9. At x = 4: y = 5. The number has tens digit 4 and units digit 5, so the number is 45 (not 54).
Q19Hard
On the same coordinate plane, graphs of y = 2 and x = −3 are drawn. What is their point of intersection?
(2, −3)
(−3, 2)
(0, 0)
(−2, 3)
Hint: y = 2 is a horizontal line; x = −3 is a vertical line. They cross where x = −3 AND y = 2 simultaneously → (−3, 2).
Q20Hard
The lines x = 1, y = 4, and x + y = 8 mutually intersect forming a triangle. Find the three vertices and the area of the triangle.
(1, 4), (1, 7), (4, 4); Area = 4.5 sq. units
(1, 4), (1, 7), (4, 4); Area = 9 sq. units
(1, 1), (4, 4), (1, 7); Area = 6 sq. units
(1, 4), (1, 8), (4, 4); Area = 6 sq. units
Hint: Intersections: x=1 ∩ y=4 → (1,4); x=1 ∩ x+y=8 → (1,7); y=4 ∩ x+y=8 → (4,4). Base = 4−1 = 3, Height = 7−4 = 3. Area = ½ × 3 × 3 = 4.5 sq. units.
